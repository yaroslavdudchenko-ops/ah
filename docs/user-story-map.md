## User Stories по приоритетам

### 🔴 P0 — MVP (обязательно, первые ~10 часов)

| ID | Роль | История | Критерии приёмки |
|---|---|---|---|
| US-001 | Исследователь | Хочу создать протокол, выбрав шаблон фазы и заполнив форму параметров | Форма с 7+ полями, валидация, сохранение в БД |
| US-002 | Исследователь | Хочу запустить AI-генерацию всех 9 секций одной кнопкой | Все секции сгенерированы, прогресс виден, время ≤ 120с |
| US-003 | Исследователь | Хочу видеть сгенерированный протокол со структурой заголовков | Секции отображаются с H2/H3 заголовками, навигация |
| US-004 | Исследователь | Хочу скачать протокол в Markdown | `.md` файл с полной структурой |
| US-005 | Исследователь | Хочу видеть список всех своих протоколов | Таблица с названием, фазой, датой |

### 🟡 P1 — Should Have (~8-12 часов после P0)

| ID | Роль | История | Критерии приёмки |
|---|---|---|---|
| US-006 | Исследователь | Хочу скачать протокол в HTML | `.html` с CSS-стилизацией |
| US-007 | Исследователь | Хочу скачать протокол в DOCX | `.docx` с заголовками Word styles |
| US-008 | Исследователь | Хочу перегенерировать одну секцию без пересоздания всего документа | Кнопка "↺" у каждой секции, создаёт новую версию |
| US-009 | Исследователь | Хочу получить проверку консистентности терминологии | Список найденных несоответствий МНН/названия |
| US-010 | Исследователь | Хочу видеть версии протокола (v0.1, v0.2) | Список версий с датой и комментарием |

### 🔵 P2 — Stretch Goals (если останется время)

| ID | Роль | История | Критерии приёмки |
|---|---|---|---|
| US-011 | Исследователь | Хочу сравнить две версии протокола | Diff: добавленное зелёным, удалённое красным |
| US-012 | Медревьюер | Хочу получить список открытых вопросов | Отдельный файл с вопросами по секциям |
| US-013 | Исследователь | Хочу получить заготовку SAP | AI-генерация SAP как отдельного артефакта |
| US-014 | Исследователь | Хочу получить заготовку ICF | AI-генерация ICF как отдельного артефакта |
| US-015 | Исследователь | Хочу видеть GCP-compliance подсказки | Список рекомендаций с ссылками на ICH E6 |
