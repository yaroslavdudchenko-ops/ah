## v0.1.0 — Initial Release

**Дата:** 2026-04-24 _(планируемая)_  
**Тип:** Initial MVP Release

### Новые функции

#### Управление протоколами
- Создание протокола с выбором шаблона (фаза I / II / III)
- Структурированная форма ввода: индикация, популяция, МНН/название препарата, primary endpoint, дозирование, критерии исключения
- Список всех протоколов с фазой, индикацией и статусом

#### AI-генерация
- Автоматическая генерация 12 разделов протокола через внутренний AI Gateway (InHouse/Qwen3.5-122B)
- Секции: Введение, Цели, Дизайн, Популяция, Дозирование, Оценка эффективности, Оценка безопасности, Статистика, Этика
- Адаптация языка под терапевтическую область и фазу
- Прогресс генерации в реальном времени (секция за секцией)
- Повтор при ошибке AI (до 3 попыток)

#### Экспорт
- Экспорт в Markdown (`.md`)
- Экспорт в HTML (`.html`)

#### Безопасность
- API ключи только через переменные окружения
- Синтетические демо-данные (BCD-100, BCD-089)
- Пометка "AI-generated draft" в документах

### Демо-данные
- Протокол 1: BCD-100 (Пролголимаб) — Фаза II, метастатическая меланома
- Протокол 2: BCD-089 (IL-17A ингибитор) — Фаза III, псориаз

### Известные ограничения
- Нет аутентификации пользователей (демо-режим)
- Максимум 120 секунд на полную генерацию протокола
- Только синтетические данные — не для реальных клинических исследований
